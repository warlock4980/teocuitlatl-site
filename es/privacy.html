Aviso de privacidad

Datos limitados, uso claro.

Project Mexica es un sitio prototipo. Recolectamos solo lo necesario para operar la encuesta, responder mensajes y mantener el sitio funcionando.

Encuesta de medallones

Cuando votas en la galería, tus medallones seleccionados se guardan en este navegador para recordar tu top tres.

El conteo respaldado por Neon puede guardar tus IDs seleccionados, datos técnicos básicos, una huella hash para reducir votos duplicados y fecha/hora del voto.

La encuesta mide interés en conceptos; no está diseñada para identificar personalmente a votantes.

Puedes eliminar los datos de este navegador en la galería con Reiniciar mi voto. Eso también solicita borrar el voto asociado a tu huella hash actual.

Chatbot

El chatbot responde muchas preguntas con una FAQ local en tu navegador. Si la FAQ no cubre la pregunta, tu mensaje puede enviarse a un proveedor de modelo de lenguaje por conexión cifrada para generar una respuesta.

Cuando se usa el modelo, la API puede guardar el texto del mensaje, un extracto de la respuesta, una huella hash técnica y fecha/hora del intercambio.

No pegues seed phrases, llaves privadas, contraseñas, documentos de identidad, datos de tarjetas ni información sensible en el chatbot.

Para borrar historial del chatbot junto con datos de encuesta de este navegador, abre la galería y usa Reiniciar mi voto.

Mapa de viajes

El Hub de Viajes usa mosaicos públicos de OpenStreetMap para mostrar marcadores y contexto. Al abrir el mapa, tu navegador puede pedir mosaicos directamente a infraestructura de terceros. Project Mexica no controla esos logs.

Correo

Si nos contactas por email, usamos tu mensaje y dirección para responder. No envíes llaves privadas, seed phrases, contraseñas o documentos sensibles por correo.

Infraestructura

GitHub Pages, Vercel, Neon y proveedores relacionados pueden procesar logs técnicos como IP, hora de solicitud, navegador y errores para entregar y proteger el sitio.

Estado de prototipo

Project Mexica es un prototipo/demo en testnet. El sitio no ofrece productos financieros, venta de tokens, moneda de curso legal, contratos de inversión ni recompensas garantizadas.
